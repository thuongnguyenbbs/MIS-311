US Baby Names Descriptive Statistics Analysis
1. Data Overview

This dataset contains information about baby names registered in the United States. The data includes four variables: Year, Name, Sex, and Count.

Year represents the year the baby name was recorded.
Name refers to the baby’s first name.
Sex indicates whether the baby is male (M) or female (F).
Count shows the number of babies given that name in a particular year.

### Key Insight

The descriptive statistics indicate that a small number of baby names dominate the overall naming records in the dataset. In particular, David and Karen show significantly higher frequencies compared to other names, suggesting that these names maintained strong popularity and cultural influence over time.

In addition, the noticeable decline in counts after the top two names demonstrates that naming preferences are highly concentrated among a limited number of familiar names. This suggests that while many baby names exist in the dataset, only a few achieve widespread and repeated use across generations.

### Key Insight

The analysis shows that female baby names accounted for approximately 53% of the total records, while male baby names represented 47%. This indicates that female names demonstrated slightly higher frequency and diversity within the dataset compared to male names.

The difference suggests that naming patterns for girls may be more dynamic and varied over time, while male naming trends appear relatively more stable and traditional. This reflects how cultural trends and modern naming preferences may influence female baby names more strongly.
